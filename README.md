# waybox
An openbox clone on Wayland (WIP)

### Goals
The main goal of this project is to provide a similar feel to openbox but on wayland

### Dependencies

*-wlroots
*-wayland

### Build instructions

```
meson build
cd build
ninja
```

After that, you should have an executable in build/
